## ТРЕБОВАНИЯ К ВМ:

### Минимальные требования:
- Python 3.8+
- 512MB RAM
- 1 vCPU
- 1GB свободного места
- Ubuntu 20.04+ / CentOS 7+ / Windows Server 2019+

### Сетевые требования:
- Открытый порт 8443 (HTTPS)
- Доступ в интернет для Telegram API
- Статический IP или DDNS

## ИНСТРУКЦИИ ПО УСТАНОВКЕ:

### Для Linux (Ubuntu/Debian):
1. sudo apt update && sudo apt install python3 python3-pip python3-venv
2. python3 -m venv venv
3. source venv/bin/activate
4. pip install -r requirements.txt
5. python telegram_bot_handler.py &
6. python https_simple_server.py

### Для CentOS/RHEL:
1. sudo yum install python3 python3-pip
2. python3 -m venv venv
3. source venv/bin/activate
4. pip install -r requirements.txt

### Для Windows Server:
1. Установить Python 3.8+
2. python -m venv venv
3. venv\Scripts\activate
4. pip install -r requirements.txt
